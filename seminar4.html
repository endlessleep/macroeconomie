Seminar 4: Shock Analysis (continued)

Expectations & AD Shocks — Mathematical impact of credibility on inflation persistence.

When expectations are "anchored," households and firms believe the Central Bank will return to its inflation target regardless of past shocks. This seminar analyzes how anchoring ($\psi < 1$) breaks the persistence of inflation compared to fully adaptive expectations ($\psi = 1$).

Exercise 1

Modifying the DAS Curve

Assume that inflation expectations follow: $$\pi_t^e - \pi^* = \psi(\pi_{t-1} - \pi^*), \quad 0 < \psi < 1$$

  1. Modify the DAS curve to incorporate this expectations mechanism.
  2. Analyze the impact of the parameter $\psi$.
Show Solution
1 — Derivation

We start from the standard Phillips curve (the supply side):

$\pi_t = \pi_t^e + \alpha(y_t - \bar{y}) + \nu_t$

Substitute the expectations mechanism $\pi_t^e = \pi^* + \psi(\pi_{t-1} - \pi^*)$ into the equation:

$\pi_t = \boxed{\pi^* + \psi(\pi_{t-1} - \pi^*) + \alpha(y_t - \bar{y}) + \nu_t}$

Or in gap form:

$\pi_t - \pi^* = \psi(\pi_{t-1} - \pi^*) + \alpha(y_t - \bar{y}) + \nu_t$
2 — Interpretation of $\psi$
  • $\psi = 1$: Fully adaptive expectations. Current inflation carries 100% of the previous gap. High persistence.
  • $0 < \psi < 1$: Partially anchored expectations. Expectations are only partially backward-looking.
  • The smaller $\psi$ is, the more expectations are anchored to the target $\pi^*$, and the faster inflation returns to target after a shock.
Exercise 2

Supply Shock Analysis ($\psi = 0.7$)

For a one-period positive supply shock $\nu_0 = 1$ with $\psi = 0.7$, calculate the path of inflation for periods 0, 1, and 2. Assume $y_t = \bar{y}$ (output stays at potential).

Show Solution
Parameters & Formula

Starting at equilibrium: $\pi_{-1} = \pi^* \implies (\pi_{-1} - \pi^*) = 0$.
Shock: $\nu_0 = 1$ and $\nu_t = 0$ for $t \ge 1$.
Simplified dynamics (assuming $y_t = \bar{y}$):

$(\pi_t - \pi^*) = 0.7(\pi_{t-1} - \pi^*) + \nu_t$
Period 0
$(\pi_0 - \pi^*) = 0.7(0) + 1 = 1 \implies \pi_0 = \pi^* + 1$
Period 1 (Shock vanishes)
$(\pi_1 - \pi^*) = 0.7(1) + 0 = 0.7 \implies \pi_1 = \pi^* + 0.7$
Period 2
$(\pi_2 - \pi^*) = 0.7(0.7) = 0.49 \implies \pi_2 = \pi^* + 0.49$

The inflation gap shrinks geometrically: $1 \to 0.7 \to 0.49$.

Exercise 3

Adaptive vs. Anchored Comparison

Compare the previous result with fully adaptive expectations ($\psi = 1$). How much faster does inflation return to target?

Show Solution
Case $\psi = 1$ (Adaptive)

Inflation inherited from last period remains identical:

  • Period 0: $\pi_0 - \pi^* = 1$
  • Period 1: $\pi_1 - \pi^* = 1$
  • Period 2: $\pi_2 - \pi^* = 1$

Without CB intervention, inflation gap stays at 1 indefinitely.

Conclusion: The Anchoring Benefit

With $\psi = 0.7$, only 70% of the previous gap is carried forward.

  • By Period 1: Gap is 0.7 (30% reduction).
  • By Period 2: Gap is 0.49 (51% reduction).

Result: Partially anchored expectations make inflation converge significantly faster.

Visualizing the convergence speed difference.

Critical Analysis Point

This entire calculation assumes that $y_t - \bar{y} = 0$. In a full Macro Model, the Central Bank would react to $\pi > \pi^*$ by raising rates and creating a negative output gap to speed up recovery.

If we ignore the CB's reaction (as in this specific exercise), inflation only returns to target if $\psi < 1$. If $\psi = 1$, the CB must intervene to stop the price spiral.
