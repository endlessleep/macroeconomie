Model Equilibrium

1. Introduction to Model Analysis

Once we have clearly defined our model structure (the IS, PC, and MPR equations), our main goal is to use this framework for analysis. Specifically, we want to understand how all the variables in the model behave when they are exposed to different changes and shocks.

A macroeconomic model fundamentally possesses two distinct types of equilibria that we must evaluate:

  • Long-run equilibrium: The ultimate destination where the economy permanently settles.
  • Short-run equilibrium: The immediate, dynamic reaction of the economy. While this state is technically not perfectly stable (because it is constantly adjusting towards the long-run), it is still formally classified as an equilibrium mathematically because all simultaneous equations temporarily balance.
Long-Run Equilibrium Conditions

The crucial first step in analyzing any economic model is to define and study its long-run equilibrium. In more complex models, economists must first make sure that a stable long-run equilibrium actually exists before they run any simulations.

How do we define long-run equilibrium in our model? The main rule is simple: everything, especially inflation, is completely stable.

Mathematically, variables no longer shift across time periods:

  • $y_t = y_{t-1}$
  • $\pi_t = \pi_{t-1}$
  • $i_t = i_{t-1}$
  • $r_t = r_{t-1}$

Logically, this leads to two main anchors: $y_t = \bar{y}_t$ (equilibrium output sits perfectly at its natural potential) and $\pi_t = E_t \pi_{t+1} = \pi^*_t$ (actual inflation equals expected inflation, which equals the central bank's target).

For these conditions to hold at the same time, the real interest rate must return to its natural rate: $r_t = \rho$. As a result, using the Fisher equation, the nominal interest rate must be: $i_t = \rho + \pi^*_t$.

This final value, $\rho + \pi^*_t$, is known as the long-run neutral interest rate.

2. Navigating the Short-Run Equilibrium

Analyzing the short-run is mathematically complex. We have 5 linked equations and 5 variables. Trying to solve this 5-dimensional system just by thinking about it is nearly impossible.

The equations are deeply interconnected: To find the output level, you need the real interest rate. To find the real interest rate, you need the nominal interest rate. And to know the nominal rate set by the central bank, you must already know the output gap. This creates a circular loop.

To resolve this, modelers employ two distinct operational solutions:

Approach 1: 2D Simplification

Mathematically combine the equations to compress the 5 variables into a simpler 2 variable space. This allows us to draw the most important parts of the model on a single 2D graph (usually Output vs Inflation).

Approach 2: Direct Simulation

Use computers and software to automatically calculate how the model responds to outside shocks, and analyze the results using multiple graphs (called Impulse Response Functions).

We will use the first approach (graphical 2D simplification) to build understanding.

3. The Dynamic Aggregate Supply (DAS) Curve

To reduce our 5 equations down to 2, we first simplify the supply side. We do this by taking the Phillips Curve and combining it with our assumption about inflation expectations.

Using purely backward-looking expectations ($E_t \pi_{t+1} = \pi_{t-1}$), we yield the Dynamic Aggregate Supply (DAS) curve:

$\pi_t = \pi_{t-1} + \phi(y_t - \bar{y}_t) + \nu_t$

Slope Intuition

The DAS curve is upward sloping. This comes from the logic of the Phillips curve: Higher output leads to more pressure on the labor market, which causes higher prices and faster inflation (the classic demand-pull mechanism).

Changes in output cause a movement along the DAS curve. On the other hand, changes in past inflation ($\pi_{t-1}$), potential output ($\bar{y}_t$), or supply shocks ($\nu_t$) will shift the entire DAS curve.

4. The Dynamic Aggregate Demand (DAD) Curve

We now combine the entire remaining structure of the model (The Demand Side, The Monetary Rule, and the Fisher conversion) into a single line: The Dynamic Aggregate Demand (DAD) curve.

Step-by-step Derivation

We use the four remaining equations:

  • IS Curve: $y_t = \bar{y}_t - \alpha(r_t - \rho) + \epsilon_t$
  • Fisher Equation: $r_t = i_t - E_t\pi_{t+1}$
  • Monetary Rule: $i_t = \pi_t + \rho + \theta_\pi (\pi_t - \pi^*_t) + \theta_y (y_t - \bar{y}_t)$
  • Expectations: $E_t\pi_{t+1} = \pi_t$

Step 1: Substitute the Fisher substitution directly into the IS Curve:

$y_t = \bar{y}_t - \alpha(i_t - E_t\pi_{t+1} - \rho) + \epsilon_t$

Step 2: Inside that resultant equation, substitute the central bank's nominal Monetary Rule for $i_t$:

$y_t = \bar{y}_t - \alpha( \pi_t + \rho + \theta_\pi (\pi_t - \pi^*_t) + \theta_y (y_t - \bar{y}_t) - E_t\pi_{t+1} - \rho) + \epsilon_t$

Step 3: Apply the firm assumption that $E_t\pi_{t+1} = \pi_t$. Notice immediately that $\rho$ cancels with $-\rho$, and $\pi_t$ directly cancels with $-\pi_t$ (this cancellation logic holds strictly because we assume the Taylor Rule perfectly reacts one-to-one to base inflation before applying its $\theta_\pi$ premium weighting).

$y_t = \bar{y}_t - \alpha( \theta_\pi (\pi_t - \pi^*_t) + \theta_y (y_t - \bar{y}_t) ) + \epsilon_t$

Step 4: Expand the $\alpha$ multiplier, and collect all like terms pertaining to the isolated variable $y_t$ onto the left-hand side:

$(1 + \alpha\theta_y) y_t = (1 + \alpha\theta_y) \bar{y}_t - \alpha\theta_\pi (\pi_t - \pi^*_t) + \epsilon_t$

Step 5: Divide the entirety by $(1 + \alpha\theta_y)$ to cleanly solve strictly for $y_t$, producing the final definitive DAD blueprint:

$y_t = \bar{y}_t - \frac{\alpha\theta_\pi}{1 + \alpha\theta_y} (\pi_t - \pi^*_t) + \frac{1}{1 + \alpha\theta_y} \epsilon_t$

This equation shows that DAD relates output only to inflation and demand shocks, assuming the central bank follows its Taylor Rule and all agents use backward-looking expectations. To make graphs easier to draw, this is often simplified as $y_t = \bar{y}_t - \gamma_1(\pi_t - \pi^*_t) + \gamma_2\epsilon_t$.

5. Analyzing DAD Slopes and Movements

The DAD curve shows a negative relationship between inflation and the total demand for goods and services. Why does it slope downwards? It comes directly from how the Central Bank reacts.

The Central Bank Mandate dictates the DAD Slope

The DAD slopes downwards because the Central Bank reacts to rising inflation by increasing nominal interest rates even faster than inflation (the Taylor Principle). This causes the real interest rate to rise, which reduces the total demand for goods.

Thought Experiment: What if the Central Bank was weak and increased nominal rates less than inflation? The real interest rate would actually fall, stimulating too much demand. The entire DAD curve would become upward sloping, causing the economy to lose its stability.

So, what controls the steepness of the DAD curve? It depends on how aggressively the Central Bank reacts. A stronger reaction against inflation (a large $\theta_\pi$ parameter compared to $\theta_y$) makes the DAD curve much steeper (more inelastic) on the graph.

A change in inflation simply causes a movement along the DAD curve. However, changes to potential output, the inflation target, or external demand shocks will cause the entire DAD curve to shift to the left or right.

6. Bringing it Together: Equilibrium Dynamics

The short-run equilibrium is found where the DAD and DAS curves intersect. Together, they determine output and inflation at the same time.

The Dynamic Form

Technically, the DAS curve is dynamic. It connects directly to previous time periods (because $\pi_t$ depends on $\pi_{t-1}$).

The Static Form

In contrast, the DAD curve is not dynamic across time periods. Its values only depend on what is happening right now, not on the past.

Critical Behavior After a Shock: After a shock hits the economy, only the DAS curve keeps shifting over time (because inflation expectations change each period). The DAD curve stays in place until a new, separate shock moves it.

The Explicit Algebraic Solution [Advanced]

We don't need to rely on graphs alone. By combining both the DAD and DAS equations algebraically, we can solve for the exact values of output and inflation as functions of shocks:

$Y_t = \bar{Y}_t + \frac{\alpha\theta_\pi ( \pi^*_t - \pi_{t-1} - \nu_t ) + \epsilon_t}{1 + \alpha(\theta_\pi\phi + \theta_y)}$

$\pi_t = \frac{(1 + \alpha\theta_y)(\pi_{t-1} + \nu_t) + \alpha\theta_\pi\phi\pi^*_t + \phi\epsilon_t}{1 + \alpha(\theta_\pi\phi + \theta_y)}$

Once we know $Y_t$ and $\pi_t$, we can use the other equations to also find the nominal rate $i_t$, expected inflation $E_t \pi_{t+1}$, and the real rate $r_t$.

Simulating an Increase in Potential Output

If we use these formulas to simulate a positive economic event, like an increase in potential output ($\bar{y}_t$), the graph shows symmetrical shifts for both the DAD and DAS curves. This symmetrical movement leads to permanent, long-term output gains, without causing any lasting change in inflation.
